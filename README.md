## 🚀 Getting Started

The website is hosted on GitHub Pages and is automatically deployed.

### Live Website
👉 **https://illustionworld.github.io**

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/illustionworld/illustionworld.github.io.git
cd illustionworld.github.io
```

2. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000` in your browser
